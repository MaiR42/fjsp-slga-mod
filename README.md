# fjsp-slga-mod
A modification for Self Learning Genetic Algorithm (SLGA) in Flexible Jop Shop Scheduling Problem (FJSP)


## How to use
* Download the instances (kacem and brandimarte)

* Execute the Makefile and run the instances
    * make
    * ./slga_mod mk01.txt mk02.txt mk03.txt mk04.txt mk05.txt mk06.txt mk07.txt mk08.txt mk09.txt mk10.txt
